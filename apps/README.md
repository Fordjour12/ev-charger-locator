# EV Charger Finder

A mobile and web application that helps EV drivers discover nearby charging stations,
bookmark reliable locations, and contribute to improving charger data through community reporting.

## Tech Stack
- React Native (Expo)
- TanStack Router
- Convex Backend
- shadcn-ui + TailwindCSS

## Features
- Map discovery of chargers
- Station details
- Bookmarks
- Suggest stations
- Report issues
- Sponsored listings




---

# Core Features

## Station Discovery

- Map view
- List view
- Nearby charger search
- Connector filtering

## Station Details

- Address
- Connector types
- Charging power
- Access information
- Pricing notes

## Bookmarks

Users can:

- Save stations
- View saved stations
- Navigate quickly

## Community Contributions

Users can:

- Suggest new stations
- Report broken chargers
- Improve station data

## Analytics

Tracks:

- Station views
- Direction clicks
- Bookmarks
- Search activity
