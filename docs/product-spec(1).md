# Product Specification

## Vision

Build the most reliable EV charger discovery platform for emerging EV markets.

## Users

EV Drivers:
- discover chargers
- bookmark locations
- navigate to chargers

Station Owners:
- claim listings
- manage station info
- sponsor listings

## MVP

- Map discovery
- Charger details
- Bookmarks
- Suggest stations
- Report issues

## Monetization

- Sponsored listings
- Station claim dashboard
- Analytics for owners
