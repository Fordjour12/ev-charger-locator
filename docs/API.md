# API Documentation

## stations.nearby

Returns nearby charging stations.

Arguments:
- lat
- lng
- radiusKm

## stations.byId

Returns station details.

## bookmarks.add

Adds station bookmark.

## bookmarks.remove

Removes bookmark.

## suggestions.submit

Submit a new station.

## reports.create

Report charger issues.

## analytics.track

Tracks events:
- station_view
- direction_click
- bookmark_add
